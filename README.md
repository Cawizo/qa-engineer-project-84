## О проекте

Проект — мануальное тестирование веб-магазина: проработка требований,
проектирование тест-кейсов, прогон, баг-репорты и регрессионное
тестирование после выхода второй версии сайта.

### Объект тестирования

- Версия 1: https://products-store.hexlet.app/cart
- Версия 2 (после доработок): https://products-store-v2.hexlet.app/

### Что сделано

- Проработал требования к функциональности магазина
- На их основе составил тест-кейсы
- Провёл тестирование по тест-кейсам и зафиксировал результаты в репортах
- Упавшие (Fail) кейсы детально описал в баг-репортах
- После выхода второй версии провёл регрессионное тестирование
- Перепроверил результаты и обновил баг-репорты

### Структура проекта

| Раздел        | Содержание                                              |
|---------------|---------------------------------------------------------|
| Требования    | Проработанные требования к сайту                        |
| Тест-кейсы    | Кейсы, составленные на основе требований                |
| Репорты       | Результаты прогона тест-кейсов                          |
| Баг-репорты   | Описание дефектов по упавшим кейсам                     |
| Регресс       | Результаты регрессионного тестирования второй версии    |
